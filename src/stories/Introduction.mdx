Tissaia-AI is a modern photo restoration application powered by Gemini Vision AI.

This Storybook documents all UI components used in Tissaia-AI, following the **Atomic Design** methodology:

- **Atoms**: Basic building blocks (Skeleton, ProgressBar)
- **Molecules**: Simple component combinations (Toast, Header)
- **Organisms**: Complex UI sections (Sidebar, UploadView)
- **Templates**: Page-level layouts
- **Pages**: Complete views
